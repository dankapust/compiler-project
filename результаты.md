## Команды для проверки

### Запуск лексера

```bash
python -m cli lex --input examples/hello.src --output tokens.txt
```

### Запуск парсера (AST)

```bash
python -m cli parse --input examples/hello.src --ast-format text --verbose
python -m cli parse --input examples/hello.src --ast-format dot --output ast.dot
python -m cli parse --input examples/hello.src --ast-format json --output ast.json
```

### Запуск тестов

```bash
python -m tests.test_runner
python -m tests.test_runner --only parser
python -m tests.test_runner --only lexer
python -m tests.test_runner --update
```

## Итог

Sprint 1–2 выполнены: **лексер + формальная грамматика + recursive descent парсер + AST + вывод (text/dot/json) + golden tests**. Дополнительно: **препроцессор** (Sprint 1 Stretch Goal), **устойчивое восстановление после ошибок** (ERR-1…ERR-4).
